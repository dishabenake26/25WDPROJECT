# 🧠 Memory Game Pro

A fun and interactive **Memory Card Game** built using HTML, CSS, and JavaScript.  
It includes animations, levels, sound effects, and a modern glass UI design.

---

## 🎮 Features

- 🎨 Beautiful glassmorphism UI design
- 🧩 Multiple levels (1–10)
- 🎯 Card matching logic
- 🎉 Confetti celebration on win
- ✨ Falling sprinkle animation effect
- 🌟 Background star effects
- 🖱️ Cursor sparkle effect
- 🔄 Smooth card flip animation
- ⏱️ Timer + move counter
- 🔊 Background music support
- 📱 Fully responsive design

---

## 🛠️ Technologies Used

- HTML5
- CSS3 (Animations + Glass UI)
- JavaScript (Game Logic & DOM manipulation)

---

## 🎯 Game Rules

- Flip two cards at a time
- If they match → they disappear with animation
- If not → they flip back
- Match all pairs to win the game
- Your score depends on time and moves
